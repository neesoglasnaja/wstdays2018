Как выйти из Vim

Отброчный тур: все против всех

Sublime Text 3

  • Начать работу очень просто
  • Действительно легкий и быстрый
  • Отлично подойдет для небольших базовых проектов
  • Всего лишь "блокнот"
  • Нет встроенного менеджера пакетов
  • Условно-бесплатный

Atom
==
Sublime Text 3

Atom

  • Начать работу очень просто
  • Действительно легкий и быстрый
  • Отлично подойдет для небольших базовых проектов
  • Всего лишь блокнот
  • Нет встроенного менеджера пакетов → Есть
  • Условно-бесплатный → Бесплатный
  • Проблемы с производительностью

Visual Studio Code

  • Подойдет под проекты практически любой сложности;
    для некоторых технологий есть много встроенных бонусов
  • Встроенная отладка Node.js, терминал и git
  • Поддержка IteliSence
  • По ощущениям более производительный, чем Atom
  • Многие функции только через плагины
  • Все еще есть баги

WebStorm

  • Может (почти) все "из коробки"
  • А что не может, есть еще доп. плагины
  • Быстрый производительный поиск по большим проектам
  • Интерфейс кажется слишком сложным
  • Для простых проектов кажется overhead
  • Проблемы производительности на не очень мощных компьютерах
  • Цена

Выводы

  • Каждый из редакторов хорош для своих задач
  • Для написания HTML/CSS/JS не должен быть нужен
    ни один из современных редакторов кода
  • Отбросьте предрассудки в выборе инструментов
  • Не бойтесь пробовать новое и искать лучшее
